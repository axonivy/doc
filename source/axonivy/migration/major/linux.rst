Linux
====================

On Linux systems the migration is very similar to a
hotfix migration. However, some additional steps are required at the end.

.. rubric:: Preparation

#. Download the latest Axon Ivy Engine hotfix version that matches your main
   release :dev-url:`/download`
#. Stop the running Axon Ivy Engine instance.
